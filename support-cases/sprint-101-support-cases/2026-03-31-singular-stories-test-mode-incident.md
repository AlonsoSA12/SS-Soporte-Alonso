# Singular Stories Test Mode Incident

## Date
2026-03-31

## Context

Several users reported that `Singular Stories` and `t-dashboard` were failing or not loading correctly.

After reviewing the incident, it was identified that a recent update had left the platform running in test mode.

---

## Actions Taken

- Reviewed reports coming from multiple Slack channels
- Validated URLs, browser behavior, and user reproduction attempts
- Considered both platform-level and `Noloco`-related possibilities
- Identified that the recent release had left test mode enabled
- Switched `Singular Stories` back to production mode
- Requested user confirmation after refresh

---

## Result

The platform was restored to production mode and users recovered normal access to `Singular Stories`.

---

## Time Spent

1 hour
