# QA Points Dashboard Visibility Issue

## Date
2026-03-31

## Context

Esteban reported that he was creating `QA points` for `J2 Academy`, but they were not appearing in the dashboard when filtered by `current quarter` or `current month`.

The issue affected the expected visibility of the newly created records.

---

## Actions Taken

- Reviewed the dashboard issue related to `QA points`
- Confirmed that the records were actually being created
- Identified a date misalignment between the day shown to the user and the date being taken by the server
- Explained the origin of the issue to continue validation
- Followed up directly with the user to confirm the behavior

---

## Result

The visibility issue was traced to the date being used by the server, which affected how the `QA points` appeared under the dashboard filters.

---

1 hour
